Biblioteca IA

Plataforma web para gestión de biblioteca escolar con catálogo digital, sistema de préstamos y reservas, y login de alumnos con Google o usuario/contraseña. El núcleo es un asistente IA integrado que permite buscar libros por título, autor, categoría o tema mediante chat. Incluye panel de check-in, reseñas y dashboard para la bibliotecaria.

python ia / chatbot google oauth roles de usuario sql

Fichero Digital con QR

Sistema web de control de asistencia para cursos escolares. Cada alumno tiene su propio código QR; al escanearlo con la app complementaria se registra automáticamente la entrada y salida calculando horas acumuladas. Reemplaza el fichero en papel y permite al docente ver el estado de cada alumno en tiempo real.

javascript qr scanner html/css base de datos mobile

Calificador de Notas

Plataforma con acceso autenticado para docentes que permite cargar y gestionar calificaciones organizadas por materia y alumno. Calcula automáticamente promedios individuales y por asignatura, facilitando el seguimiento académico y el cierre de trimestres sin planillas manuales.

javascript html/css acceso docente promedios automáticos sql
