sobre mí

Soy Lionel Medina, desarrollador web con formación técnica especializada en programación. Me dedico a construir aplicaciones web completas que resuelven problemas reales para usuarios reales.

Tengo experiencia trabajando en el stack completo: autenticación de usuarios, bases de datos, APIs, interfaces responsivas e integración de inteligencia artificial. Cada proyecto que desarrollo tiene un objetivo claro y está pensado para ser escalable y mantenible.

Me interesa especialmente crear productos que sean tan sólidos técnicamente como simples de usar. El código limpio y la buena experiencia de usuario no son opuestos.

Biblioteca IA

Sistema de gestión de biblioteca escolar con catálogo digital, préstamos y reservas de libros, y autenticación de usuarios con Google OAuth. Su funcionalidad principal es un asistente de inteligencia artificial integrado que permite buscar libros por título, autor, categoría o temática mediante conversación en lenguaje natural. Incluye roles diferenciados para alumnos y bibliotecaria, panel de check-in, reseñas y dashboard administrativo.

Python IA / Chatbot Google OAuth SQL Roles de usuario Render

Fichero Digital con QR

Solución web para el control de asistencia en cursos técnicos. Cada alumno cuenta con un código QR único que es escaneado por una app complementaria al momento de entrada y salida, registrando automáticamente las horas acumuladas por jornada. Elimina el uso del fichero en papel y permite al docente visualizar el estado de asistencia en tiempo real desde cualquier dispositivo.

JavaScript HTML / CSS QR Scanner Control de horas Responsive

Calificador de Notas

Plataforma de gestión académica orientada a docentes con acceso autenticado. Simplifica el cierre de trimestres y el seguimiento del rendimiento académico sin necesidad de planillas manuales ni hojas de cálculo externas.

Turnos · Consultorio Psicóloga

Plataforma web para la gestión de un consultorio psicológico privado. Los pacientes pueden registrarse con Google o correo electrónico, reservar turnos de forma online y comunicarse directamente con la psicóloga mediante un chat en tiempo real. La profesional cuenta con un panel de administración exclusivo para gestionar su agenda. Desplegada en producción en Vercel.
